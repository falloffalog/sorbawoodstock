# SORBA Woodstock

An Alexa Skill/App for easy lookup of SORBA trail status.

## Built With

* [Alexa Skills Kit](https://developer.amazon.com/alexa-skills-kit)
* [AWS Lambda](https://aws.amazon.com/lambda/)
* [Python](https://www.python.org/)
